About the Rating System

üéØ Methodology

This leaderboard uses a Glicko rating system combined with session-based z-score evaluation to rank World vs World performance.

üìä How It Works

  • Session-Based Evaluation: Each combat session is analyzed independently
  • Z-Score Calculation: Player performance is compared to others in the same session
  • Glicko Rating: Dynamic rating system that accounts for uncertainty and volatility
  • Composite Scoring: Combines Glicko rating with average rank performance

üèÖ Leaderboard Types

  • Overall: Combined ranking across all metrics
  • Individual Metrics: Specialized rankings for DPS, healing, support, etc.
  • Profession-Specific: Role-based rankings with weighted metrics

üìà Key Metrics

  • Composite Score: Final ranking score combining multiple factors
  • Glicko Rating: Base skill rating (higher = better)
  • Avg Rank%: Average percentile rank in sessions (lower = better)
  • Games: Number of combat sessions analyzed
